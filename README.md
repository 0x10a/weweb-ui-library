# WeWeb UI Library

Shadcn-inspired UI component library for WeWeb.

## Components

- **Button** - Primary, Secondary, Outline, Ghost, Destructive variants
- **Input** - Text, Email, Password, Number with validation
- **Textarea** - Multi-line text input
- **Select** - Dropdown select with search
- **Checkbox** - Single and group checkboxes
- **Radio** - Radio button groups
- **Switch** - Toggle switch
- **DatePicker** - Date selection with calendar

## Installation

```bash
npm install
```

## Development

```bash
npm run serve
```

## Build

```bash
npm run build
```

## Design Tokens

All components use consistent Shadcn design tokens defined in `src/styles/tokens.css`.
